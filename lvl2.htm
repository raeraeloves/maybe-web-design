Taste is to describe beauty by your senses.

Good design relates moods and feeligs.

Web design should be fluid, to adapt to a variety of devices.

First impressions are vital.

The aesthetics of a design can be the difference between positive and negative brand association.

Visual appeal can be immediate and unconscious.

Using Elements of Design

Understanding the Elements of Design

Use these as guides in aesthetic decision making.

Elements of design: The basic components involved with creating any design, whether it's for print or web.

Principles of Design: The rules of design, which should always be considered even if you end up not using them.

Principles are the rules, and elements are the things that help you follow the rules.

There are 8 elements of design: color (hue), value (tone), texture, shape, form, space (whitespace), line, type

Using Color to Set the Site's Mood

Set the right mood and convey a deliberate feel or emotion that supports the information within the design.

Primary Colors: red, blue, yellow

Secondary Colors: violet, green, orange

Tertiary Color - a combination of two tertiary colors

Adding white to a color creates a tint.

Adding black to a color creates a shade.

Color harmonies are combinations of colors which people tend to find aesthetically pleasing.

Monochrome: a single color and all of its iterations from white to black

Complementary Colors: represent a par of colors at opposite ends of the color wheel like red and green.

Split Complementary Colors: Selecting a color, and the two colors adjacent to that color's compliment, like red, yellow-green, and green-blue

Analogous (adjacent) Colors: any two colors that are directly next to each other on the color wheel

Color Triad: any three colors equally spaced on the color wheel, like red, green and blue

Colors can represent and convey emotion.

Think about which colors will best represent the product or service you're designing for and which colors will appeal most to the target audience.

Think about the mood you want to project, and use your color wheel to check for harmony.

Tweaking Color Value to Add Contrast

Value refers to the relative lightness or darkness of a color within its composition.

Value can help to convey emotion and meaning as well as adding contrast.

Value is the degree of lightness or darkness in a given color.

A pure color plus gray is a tone.

Pastels: happiness, femininity, babies, spring

Muted Tones: sadness, masuclinity, cooler months, serious affairs, camping

Creating contrast can set focal points or indicate areas of emphasis.

Any light object on a dark background is the first place your attention goes, and becomes the focus of attention. Bigger gets more.

The same thing happens with light objects on dark and bright objects on neutral.

Gradation and shadowing can give the illusion of depth.

Using Texture to Add Depth

Flat design is a current trend, but texture can create interest and depth.

Texture: the surface quality of any shape within your web layout

Think of how a texture would look and feel and consider how they are created.

Photograph/ scan textures, create textures using filters and blending modes, 'paint' texture into design, use stock images

Where to get texture online: smashingmagazine.com/tag/textures, textureking.com, lostandtaken.com, texturemate.com, zentextures.com, textures.com

Texture can help emphasize a focal point or be used as a decorative element.

Textures can be combined with shapes, lines, colors, lighting, and typography.

Add a sense of depth, using a reflective surface.

Use textures as backgrounds, also to add depth.

Textures can help create a mood.

Combining textures can delineate different areas on your web page.

Texture can go a long way toward making a site memorable and unique.

Repeating Shapes to Unify Your Design

Shape can help set the tone of web graphics.

Repeating shapes within a design can help unify all elements in a single layout.

Shape is any flat area bound by line, our outline, value or color.

Shapes are usually self-contained defined areas which can be positive (occupy space) or negative (like whitespace, the area between positive shapes).

In web design, shapes can be functional elements like buttons and navigation bars.

Shapes can be used to carve space, organize, and define areas.

Shape repetition can create a sense of harmony or unity in your design.

Structuring Your Layout with Form

Form is the overall mass of shapes within your design and how they relate to one another.

Consider the group of shapes as a whole.

Everything should look like it's meant to go to gether, with nothing out of place.

Plan and execute designs in black and white before adding color.

Be playful. Incoroporate interactive elements.

Form is a way to create a balanced, harmonious layout.

Be organized. Be innovative. Keep it interesting.

Using Whitespace to Organize Your Design

Positive and negative space is created automatically by elements of your design.

Space, or whitespace, refers to the intervals, areas, or measurable distances between all the objects or elements in a design.

It's important to leave some spaces completely blank so that the eye has a place to rest between and around the elements, and the user can distinguish important information.

Use paralax scrolling to keep your user engaged.

Vary the space between pages to keep visitors engaged and interested.

Space should be used deliberately.

Setting Boundaries with Lines

Lines can define and enhance shapes within the layout and set boundaries between objects within a page.

Line refers to any actual or implied mark, path, mass, or edge which defines position and/or direction within the design.

Lines can be created using any mark making tool and can be created in nearly any shape or size.

The edge of two shapes forms a line, or the illusion thereof.

Lines can create areas of emphasis when they visually 'point' to a focal area.

Lines can imply movement by their inherent directionality and lead your eye across the page.

Use lines to organize information by defining spaces.

Use lines thoughtfully to communicate a sense of order.

Communicating With the Right Fonts

Selecting the right fonts in the right sizes and colors can help improve readability and communicate information more effectively.

Type refers to the fonts you choose for displaying text in your designs and how you actually use them.

Fonts can communicate mood, and fonts from the same typeface along with a contrasting font can keep the design cohesive.

Typefaces are a set of fonts in a given font family.

Parts of fonts: x-height, ascender, descender, stem, tail, serif

Two basic font types: serif and sans-serif

Serifs are the little hooks on letters like in Times New Roman.

Serif fonts can lead the eye from one letter to the next, but they can be difficult for dyslexic people to read and become an accessibility issue.

Sans-Serif are a preferred body font.

The leading (pronounced 'ledding') refers to the space between the baseline of one line of text and the baseline of another. In CSS this is called 'line height'.

By default, line height is usually set to 120% of the font size, but this can be overriden.

Adjusting line height can make a design feel more open.

Use consistent font formatting across all of your site's formatting so it is easy to understand at a glance.

Incorporating Principles of Design

Understanding the Principles of Design

The Elements of Design are the basic components involved with creating any design whether it's for print or web.

The Principles of design are the rules of design, which should always be considereed even if you end up not using them.

10 Principles of design: contrast, emphasis, balance, unity, pattern, movement, rhythm & repetition, proportion, simplicity, and gradation

Using Contrast to Set Areas of Interest

Contrast: the juxtaposition of opposing elements by way of using different colors, tones, directions, lines, shapes, etc. to show emphasis

When designing your site, decide in advance the relative importance of each element in your design. Which should be highlighted?

Use large elements by small elements, opposing colors, different tones or values, use horizontal, vertical, diagonal elements, play with lines, work with hard or soft edges, harmonious font pairings.

Using contrast correctly will show visitors where to look and keep them engaged.

Applying Styles to Show Emphasis

Emphasis happens when attention is focused on a single area within a design to assist the viewer in understanding visually that this particular area is more important than other areas.

Use size and placement of key elements.

Typographic Hierarchy: the specific way of using fonts to create a sense of flow and emphasis on a page by varying the size, weight, color, and spacing of text.

Make sure you understand how HTML and CSS work together.

HTML marks key areas of text before CSS is applied.

CSS styles can be made for each type of text element. There are styles for every type of text including not only headers and paragraphs, but lists, tables, forms, buttons, icons, and images.

CSS allows you to convey meaning through font, text size, bold/ italics, color, etc.

Visit the Base CSS site.

Pair Serif and Sans-Serif for heading and paragraph.

Use size, case, and color to emphasize text.

Aligning Objects to Achieve Balance

Balance occurs in a design when the elements as a whole have a feeling of equality of weight, attention, or attraction.

To achieve balance in design, one must consider the plaement and alignment of every single element in one's layout.

Balance can be created symmetrically or asymmetrically.

Balance should achieve an overall sense of harmony.

Try using the grid system to easily create a balanced layout. The elements will be more easily spaced.

Balance text areas and photographs.

Consider alignment of elements.

Using Hyperlink Styles to Create Unity

Unity = 'harmony'

Unity refers to the pleasing arrangement of parts within a composition.

Unity is the relationship between the individual parts and the whole of the design, used creatively to express a particular idea or emotion.

Unity in design presents information in a logical structure and creates a sense of order with engaging visuals.

Consider the unity of fonts, colors, shapes, lines, photos, textures, and other decorative elements.

Consider the appearence and behavior of your links and hyperlinks. Override default link settings using CSS (Link, Visited, Hover, Active).

KISS but don't be timid.

Applying Background Patterns

Patter offers a way to keep design organized while providing a sense of harmony through use of repeating elements.

Creating elements that are the same size, shape, or have equal spacing are different ways to create patterns.

Use a grid system for organization.

Patterns can be used as decorative elements, like backgrounds.

When adding patterns to your background, apply the style to the body tag of your page. You can also apply it to the wrapper.

The wrapper is the containing element that holds your desing in place within the center of the layout.

Create patterns by scanning fabrics or materials into your computer or drawing designs in Illustrator or PhotoShop.

Adding Scrolling and Animation

Movement in graphic design refers to the suggestion of action or direction. Movement in web design can include actual movement.

Movement can be created by line, texture, and shape.

The default scrolling style is 'scrolling' and travels with the text.

You can also use 'fixed' scrolling, in which the background image styas put and the content scroll on top of it.

'Parallax scrolling' is a type of scrolling in which the background images shift slower than the content in the foreground, creating an illusion of three-dimensional depth.

'Easing' is a jQuery animation effect. jqueryui.com has examples of all kinds of effects and widgets you can use with jQuery.

Make sure you are designing your effects with mobile users in mind. Check out jquerymobile.com

Movement creates a sense of interactivity for the site visitor.

Using Border Styles to Add Rhythm

Rhythm & repetition refers to the repetition of an element in a design that creates a visual rhythm.

In design, underlying structure can be paired with free-flowing deisng elements.

Rhythm helps establish visual interest in a design, and can happen naturally when the same shapes and patterns are found throughout your site.

Regular rhythm consists of even intervals flowing together.

Progressive rhythm consists of elements which grow or shrink in some way such as size, position, or color.

Try varying repetition styles to avoid monotony.

Apply border styles to any number of sides on a set of elements in your design using CSS

Border styles also help to create structure and define certain areas, and to anchor headings or subsections within an article.

Achieving Proportion by Scaling Objects

Proportion can bring a sense of balance, emphasis, and harmony to a design.

Proportion deals with the visual relationship between two or more things in terms of their size, number, or degree.

Proportion = tiered dominance, creating a sense o hierarchy and order

Tier the size of fonts to help viewers identify the important parts.

Create proportion with size by scaling the size of shapes, tiering buttons and thumbnails by 10 or 20% increments.

Using a lot of space with a little text is a good way to do this, whether that space is a large image or lots of whitespace.

Make the most important thing largest, then scale down.

Creating Simplicity

Simplicity deals with the ELIMINATION of any non-essential elements or details to reveal the essence of a form.

More simplistic designs tend to receive higher ratings for attractiveness.

Everything in your layout should be needed, and nothing is extra.

Lead the viewer through the layout, considering whether each design element is essential or non-essential.

Also ask if there is any information or elements that are missing.

Set aside a 'finished' design for a couple days before reviewing and finalizing it with fresh eyes.

Using Gradation to Create Perspective

Gradation can add a sense of interest or movement to your design.

Gradation refers to any gradual change in a design that occurs by a series of stages, steps, or degrees.

Gradation can be applied to any element within your site.

Gradation draws the eye in and creates a sense of movement in that a gradual change builds a path the eye can easily follow.

Web Design Industry and Trends

Emphasis: Designing for Hand Gestures

The average app employs 7-18 different touch gestures in addition to manipulating the device itself such as shaking, moving, and tilting it.

7 Common Gestures: tap, double-tap, press, flick, drag, swipe, spread, pinch

To use gestures effectively, interface elements need to be large enough to use

The actual sizes of interactive elements and the use of whitespace are vital to helping users avoid touch errors.

The target element should be 10 to 12 mm which is roughly 37 to 45 px

There should be at least 8 px space between elements.

Keep the thumb zone in mind. This is the area that one and two handed users are likely to reach comfortably by thumb.

Put the more important elements in the thumb zone, less important in the area under the thumb 'bend zone', most important in the area in front of the thumb.

Least important elements should be in the extending thumb area at the top of the screen.

Use whitespace to maximize touchability of important elements.

Tappable areas should be large enough for the average index finger.

Keep menus short and sweet with easy homepage access.

Make site search visible.

Use mobile-specific features like tap to call and social sharing.

Put labels above forms so they are visible as users input data.

Place calls to action in prominent view.

Try to keep load time short by using CSS elements instead of graphics.

Make sure your sliders are easy to drag.

Typography: Choosing and Using Web Fonts

Web-Safe fonts must be cross-platform, already installed on visitor's device. This list is actually rather small.

Web-Safe fonts are specified in CSS as a "font set" so that unavailable fonts have specified fallbacks. {font-family: Verdana, Arial, Helvetica, sans-serif}. The site will use the first available on the list from left to right.

Web Fonts are platform independent, do not need to be installed on visitor's device, and have hundreds of options.

Google fonts require a rule in your CSS and a link in the head to the web fonts source. There is also an option for using the @import rule, or JavaScript instead of the standard link.

Get your online fonts from subscription-based or free websites that host fonts.

Font types: serif and sans-serif, slab serif, cursive, handwriting or script, decorative, and typewriter

Choose fonts that contrast one another, play with scale, match text with tone, go BIG with scripts and slabs, and BE COURAGEOUS.

See speciman on Google's font directory offers suggestions for font pairings.

Try overlapping text, inline fonts, lettercase, and flat shadows.

Remember fonts are there to help convey meaning in a harmonious way.

Balance: Editing Images for Social Media

Balance occurs in a design when the elements as a whole have a feeling of equality of weight, attention, or attraction.

To project a unified brand image, a business must have a strong profile pic or avatar, and an equally interesting header or graphic (banner or cover image).

Profile pic should be a centered company logo or mascot. For small businesses, a photo of the business owner is also common practice.

The image should be easily discernable from its maximum size down to 25 px square.

Profile pictures: brand recognition, scalability

Headers: cropping, scaling, orientation, balance

Use the largest of multiple proportion requirements for scalability, keeping important elements centered.

Try to keep file sized under 2 MB

When creating graphics for social media, use content-aware scaling or content-aware fill to expand a background as needed to fill the frame and add marketing copy in areas where it won't be obscured by a profile pic or other overlaid text.

Color: Theory and Trends

Great design uses the colors that are right for the project, regardless of trend.

Determin the aesthetic preferences of the target audience.

Use your color wheel.

Pattern and Gradation

Pattern is a way to keep design organized while providing a sense of harmony through use of repeating elements.

Gradation is a gradual change within an element that occurs by a series of stages, steps, or degrees.

Radial gradients can creat a vignette effect that highlights content.

CSS3 gradients have soft, feathery transitions, or hard edges. Use a tool like CSSmatic.com to save time.

SVG code takes an SVG image and allows the file to be specified as the background image of a CSS element. SVG is very crisp on Hi-Rez screens and it is supported by more browsers, including IE9.

Animate a color fade using CSS and JavaScript.

Use a CSS pattern where the browser draws the pattern based on CSS rules applied to an element via a class attribute.

Use CSS to create pattern using a set of linear gradient and background color style rules.
